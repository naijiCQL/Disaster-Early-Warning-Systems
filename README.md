# D-W-System

## 项目简介

1、本项目采用Vue3 + Vite + Element Plus + pinia + Echarts进行编写，地图展示方面采用的是OpenLayers 6，主要目的是熟悉对Vue3以及OpenLayers的使用。

2、本项目作为练习实践项目，目前项目正在搭建中，后续再进行更新。

3、欢迎大家一起沟通交流。

## 项目启动

### 依赖安装

> npm install

### 启动项目

> npm run dev
